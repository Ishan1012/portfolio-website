# Portfolio Website

This is my personal portfolio website showcasing my projects, certifications, and skills. It highlights the work I've done and the learning achievements from platforms like Coursera, Udemy, NPTEL, and others. The site also features a skills section where I list my current competencies to stay updated with industry trends.

## 📦 Technologies

- `Next.js`
- `TailwindCSS`
- `Framer-motion`
- `Lucide-react`

## 👨‍🎓 The Process

This project started as a way to create a professional online presence. I wanted to build a responsive, modern website that effectively displays my background, projects, and expertise. The journey involved learning Next.js for React-based web development, integrating animations with Framer Motion, and styling with Tailwind CSS.

## 🏗️ How I Built It

Building this portfolio was a step-by-step process to ensure a clean, user-friendly experience:

### Phase 1: Planning and Design
I outlined the key sections: Hero, About, Skills, Projects, Certifications, and Contact. I focused on responsive design to work well on all devices.

### Phase 2: Development
Using Next.js, I created reusable components for each section. I implemented custom hooks for managing data like projects and certifications. Tailwind CSS was used for styling, and Framer Motion added smooth animations.

### Phase 3: Content Integration
I populated the site with real data from my projects and certifications. The skills section was designed to be easily updatable.

### Phase 4: Deployment
The site is ready for deployment on platforms like Vercel or Netlify for easy access.

## 🪧 Running the Project

To run the project locally:

1. Clone the repository to your local machine.
2. Navigate to the `client` directory.
3. Run `npm install` to install dependencies.
4. Run `npm run dev` to start the development server.
5. Open your browser to `http://localhost:3000`.
